# UNIT 5 – NLP APPLICATIONS PROJECT  
## Ben 10 Themed Omnitrix AI System  

## Overview  
This project implements core Natural Language Processing (NLP) applications using Python and Hugging Face Transformers.  
The system is designed using a Ben 10 theme, where the Omnitrix acts as an intelligent AI system performing multiple NLP tasks.

## Modules Implemented  

### 1. Chatbot – Omnitrix AI  
Simulates an intelligent conversational assistant.  
- Accepts user queries  
- Generates responses  
- Mimics human-like interaction  

### 2. Question Answering – Knowledge Core  
Answers questions based on a given context.  
- Input: Context and Question  
- Output: Extracted answer  

### 3. Text Summarization – Report Analyzer  
Summarizes long text into a concise version.  
- Reduces content length  
- Preserves key information  

### 4. Machine Translation – Galactic Translator  
Translates text from English to another language.  
- Example: English to French  

## Technologies Used  
- Python  
- Jupyter Notebook  
- Hugging Face Transformers

## How to Run  

1. Install dependencies:
```
pip install transformers
```

2. Open Jupyter Notebook:
```
jupyter notebook
```

3. Run all cells:
```
Kernel → Restart and Run All
```
